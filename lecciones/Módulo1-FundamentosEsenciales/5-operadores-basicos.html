Introducción a los Operadores Básicos

Los operadores en JavaScript son símbolos que permiten realizar operaciones sobre valores y variables. En esta lección, aprenderás sobre los operadores aritméticos, de asignación y de comparación.

Operadores Aritméticos

Los operadores aritméticos se utilizan para realizar operaciones matemáticas: 

let a = 10;
let b = 5;

console.log(a + b); // Suma: 15
console.log(a - b); // Resta: 5
console.log(a * b); // Multiplicación: 50
console.log(a / b); // División: 2
console.log(a % b); // Módulo (resto de la división): 0
console.log(a ** b); // Exponenciación: 100000

Operadores de Asignación

Los operadores de asignación se utilizan para asignar valores a las variables: 

let x = 10;
x += 5; // Equivalente a x = x + 5
console.log(x); // 15

x -= 3; // Equivalente a x = x - 3
console.log(x); // 12

x *= 2; // Equivalente a x = x * 2
console.log(x); // 24

x /= 4; // Equivalente to x = x / 4
console.log(x); // 6

Operadores de Comparación

Los operadores de comparación se utilizan para comparar valores: 

let num1 = 10;
let num2 = "10";

console.log(num1 == num2);  // Igualdad (true)
console.log(num1 === num2); // Igualdad estricta (false)
console.log(num1 != num2);  // Desigualdad (false)
console.log(num1 !== num2); // Desigualdad estricta (true)
console.log(num1 > num2);   // Mayor que (false)
console.log(num1 < num2);   // Menor que (false)
console.log(num1 >= num2);  // Mayor o igual que (true)
console.log(num1 <= num2);  // Menor o igual que (true)
