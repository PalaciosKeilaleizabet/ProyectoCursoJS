Introducción a la Conversión de Tipos

En JavaScript, la conversión de tipos (también llamada coerción) es el proceso de convertir un valor de un tipo a otro. Esto puede ocurrir de manera implícita (automáticamente) o explícita (manualmente).

Coerción Implícita

La coerción implícita ocurre cuando JavaScript convierte automáticamente un tipo de dato en otro. Por ejemplo: 

let numero = 42;
let texto = "El número es " + numero; // Coerción implícita de número a string
console.log(texto); // "El número es 42"

Coerción Explícita

La coerción explícita ocurre cuando el programador convierte manualmente un tipo de dato en otro. Por ejemplo: 

let texto = "123";
let numero = Number(texto); // Coerción explícita de string a número
console.log(numero); // 123

Métodos de Conversión

  • Number(): Convierte un valor a número.
  • String(): Convierte un valor a cadena de texto.
  • Boolean(): Convierte un valor a booleano.
  • parseInt() y parseFloat(): Convierten cadenas a números enteros o decimales.
