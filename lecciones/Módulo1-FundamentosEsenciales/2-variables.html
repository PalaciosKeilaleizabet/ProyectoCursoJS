Introducción a Variables y Constantes

En JavaScript, las variables y constantes se utilizan para almacenar datos. Las variables pueden cambiar su valor, mientras que las constantes no.

Declaración de Variables

  • let: Declara una variable que puede cambiar su valor.
  • const: Declara una constante cuyo valor no puede cambiar.
  • var: Declara una variable con ámbito de función (no recomendado en código moderno).

Ejemplo de Código

// Declaración de variables
let nombre = "Juan";
const edad = 25;

// Reasignación de variables
nombre = "Carlos"; // Correcto
// edad = 30; // Error: No se puede reasignar una constante

console.log(nombre); // Imprime "Carlos"
console.log(edad);   // Imprime 25
