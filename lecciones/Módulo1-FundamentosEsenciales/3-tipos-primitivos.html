Introducción a los Tipos de Datos Primitivos

En JavaScript, los tipos de datos primitivos son valores que no son objetos y no tienen métodos. Son inmutables, lo que significa que no pueden ser modificados después de su creación.

Tipos de Datos Primitivos

  • String: Representa una cadena de texto. Ejemplo: "Hola, Mundo".
  • Number: Representa un número, ya sea entero o decimal. Ejemplo: 42 o 3.14.
  • Boolean: Representa un valor lógico: true o false.
  • null: Representa la ausencia intencional de valor.
  • undefined: Representa una variable que no ha sido asignada.
  • Symbol: Representa un valor único e inmutable, usado como identificador de propiedades de objetos.
  • BigInt: Representa números enteros de tamaño arbitrario.

Ejemplo de Código

// Ejemplos de tipos primitivos
let nombre = "Juan"; // String
let edad = 25; // Number
let esEstudiante = true; // Boolean
let direccion = null; // null
let telefono; // undefined (no se ha asignado un valor)
let id = Symbol("id"); // Symbol
let numeroGrande = 1234567890123456789012345678901234567890n; // BigInt

console.log(typeof nombre); // "string"
console.log(typeof edad); // "number"
console.log(typeof esEstudiante); // "boolean"
console.log(typeof direccion); // "object" (¡esto es un error conocido en JavaScript!)
console.log(typeof telefono); // "undefined"
console.log(typeof id); // "symbol"
console.log(typeof numeroGrande); // "bigint"
