Introducción a la Sintaxis de JavaScript

JavaScript es un lenguaje de programación que sigue reglas específicas de sintaxis. Estas reglas definen cómo se escriben y estructuran los programas.

Reglas Básicas

  • Comentarios: Usa // para comentarios de una línea y /* ... */ para comentarios de varias líneas.
  • Punto y coma: Aunque no siempre es obligatorio, es recomendable usar ; al final de cada declaración.
  • Uso de llaves: Las llaves {} se usan para definir bloques de código.
  • 'use strict': Es una directiva que habilita el modo estricto, lo que ayuda a escribir código más seguro y optimizado.

Ejemplo de Código

// Esto es un comentario de una línea
let mensaje = "Hola, Mundo!"; // Declaración de una variable
console.log(mensaje); // Imprime el mensaje en la consola
